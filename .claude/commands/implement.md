# Implement

## Phase 1: Understand
1. Read the plan completely.
2. Verify no blockers. If issues, stop and ask.

## Phase 2: Execute
1. Follow Steps in order.
2. After each step, verify it worked before moving on.
3. If a step can't be completed, note the issue and adapt or ask.

## Phase 3: Validate
1. Check each Success Criterion.
2. Run tests if applicable.

## Phase 4: Update
1. Change plan status to `Implemented`.
2. Update `memory/active-tasks.md`.
3. Create session file and update TIMELINE.md.

## Report
Output: what was done, files changed, validation results, any deviations.
